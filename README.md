# Portfolio: Software Development & Algorithm Challenges

大学の授業課題として取り組んだ、プログラミング言語・データ構造・アルゴリズムに関する制作物のまとめです。

## 1. RobotPuzzle (Java / 2D Tile Game)
ロボットをゴールまで誘導する2Dタイルパズルゲームです。
- **概要**: 顔の向き（進行方向）を考慮しながら、ギミックを解いて最短ルートでゴールを目指します。
- **技術要素**: Java Swing を用いたGUI実装、進行方向と座標管理のロジック構築。
- **詳細資料**: リポジトリ内の `2dgame.pdf` に設計書と詳細仕様をまとめています。

## 2. Prolog Lexer (OCaml / Functional Programming)
OCamlを用いて構築した字句解析機です。
- **概要**: Prolog風の構文を解析し、論理的なクエリを処理できる基礎を構築しました。
- **応用例**: レポート内では「磯野家の家族構成」を知識ベースとして扱い、家族関係の問いに答えるロジックを実装しています。
- **技術要素**: 再帰下降構文解析、OCamlによる関数型プログラミング。

## 3. SNS Infrastructure (C Language / Data Structure)
数万件規模のユーザーデータを高速に扱うSNSのバックエンド基盤です。
- **概要**: 大規模な「フォロー関係」や「ユーザー情報」を、メモリ効率良く、かつ高速に検索・管理するシステム。
- **技術要素**: 
  - **ハッシュテーブル**: ユーザーIDの一致検索を $O(1)$ に近づける高速なデータ管理。
  - **有向グラフ**: フォロー・フォロワーの関係性をグラフ構造で表現。
- **特徴**: 数万件のデータに対しても、ストレスのない検索パフォーマンスを実現。
